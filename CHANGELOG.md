# Changelog

## 0.2.0

- Add hook integration for real-time session tracking via Unix domain socket
- Add terminal app selection for "Open in Terminal" context menu
- Add settings view for configuration

- Initial release
- Menu bar integration with status indicator (stopped/watching/active)
- Real-time Claude Code process detection

- Project cards with expand/collapse functionality
- Subagent tracking with running/completed status
- Task checklist with pending/in_progress/completed status
- Titlebar status indicator with pulse animation
- Context menu actions (Open in Terminal, Copy Path)
- About window with version info
